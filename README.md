# FKG (Fælleskommunal Gateway)
This repository serves as the technical issue tracker for development for the FKG project (Fælleskommunal Gateway, formerly known as 'KL Gateway'). Users of the FKG project being the municipality patient journal vendors are encouraged to create an issue in this repo if a problem is encountered when sending information to the FKG. For general documentation of the FKG please visit https://ehealth-dk.atlassian.net/wiki/spaces/EDTW/pages/2189361157/Care+Gateway
